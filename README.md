# ⚡ Smart Daily Task Planner

A high-performance, mobile-responsive task management application built with **React**, **Tailwind CSS**, and **Framer Motion**. Featuring an AI-powered "Smart Sort" engine and a stunning modern dark mode interface.

![Smart Daily Task Planner](https://img.shields.io/badge/Status-Active-brightgreen)
![Tech](https://img.shields.io/badge/Tech-React%20%7C%20Tailwind-blue)

## ✨ Features

- **⚡ Smart Sort Engine**: Automatically prioritizes tasks based on priority (High > Medium > Low) and due dates.
- **📊 Interactive Progress**: Visualize your productivity with a dynamic circular completion tracker.
- **🌙 Premium Dark Mode**: "Electric Blue" accents with a sleek glassmorphism aesthetic.
- **📱 Fully Responsive**: Optimized for both mobile and desktop views.
- **💾 Local Persistence**: All tasks are saved automatically to `localStorage`.
- **🎭 Smooth Animations**: Powered by Framer Motion for a premium user experience.

## 🛠️ Built With

- **React**: Component-based UI logic.
- **Tailwind CSS**: Modern utility-first styling.
- **Framer Motion**: Fluid micro-animations.
- **Lucide Icons**: Crisp, professional iconography.
- **Babel**: On-the-fly JSX transpilation.

## 🚀 Getting Started

Since this is a single-file application, you don't need to install any dependencies!

1. Clone the repository:
   ```bash
   git clone https://github.com/TalatNoman/Smart-daily-task-planner.git
   ```
2. Open `index.html` in your favorite browser.
3. Start planning your day!
